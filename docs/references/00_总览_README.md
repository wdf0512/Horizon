# Horizon 个人变现路径 · 三方向执行 Spec 总览

## 我是谁、要做什么

我是 Horizon(https://github.com/Thysrael/Horizon)这个开源 AI 新闻聚合项目的**个人用户**,不是项目所有者。我用下来体验非常好,想以这个工具为生产引擎,从 0 开始建立自己的副业/事业。本目录的 3 个 spec 是 3 条独立的执行路径,每一条都可以单独启动,也可以组合启动。

## 三个方向对比

| 维度 | 方向一:中文垂直日报 | 方向二:英文专业垂直日报 | 方向三:个人研究助手 |
|---|---|---|---|
| **核心产出** | 中文 AI 日报订阅 | 英文专业垂直 newsletter | 个人深度文章/咨询/小报告 |
| **启动成本** | 极低(API 费 ¥1-3/天) | 中(API $5-10/天) | 几乎零 |
| **变现速度** | 6-12 个月 | 3-9 个月 | 1-3 个月 |
| **收入天花板(12 个月)** | $1000-3000/月 | $3000-10000/月 | $2000-5000/月 |
| **核心风险** | 中文 newsletter 天花板低 | 需要行业理解力 / 冷启动难 | 需要写作能力 |
| **护城河** | 信息源 taste + 摘要审美 | 行业垂直认知 + 受众网络 | 个人判断力 + 输出质量 |
| **是否需要被看见** | 是(订阅模式) | 是(订阅 + 销售) | 半是(只需小范围声誉) |
| **适合什么样的人** | 有耐心、愿意运营内容 | 有目标行业、有沟通能力 | 有写作能力、想轻资产起步 |

## 推荐启动顺序

**第一优先级:方向三**

零风险零成本,做的过程会自然告诉你"我对哪个垂直最有判断力",为方向一/二 选垂直提供输入。

**第二优先级:方向一**

在做方向三的第 4-8 周可以同时启动方向一。两个方向有内容协同:方向三深度文章可以成为方向一日报的"特色专栏"。

**最后考虑:方向二**

等方向一/三 跑了 3-6 个月、有了读者反馈和行业判断力后再启动。或在方向三中发现某个英文垂直明显有商业空间时启动。

## 三个 spec 文件的使用说明

每个 spec 文件都是一份**给 AI 助手的长期项目协作 prompt**。使用方式:

1. 选定要启动的方向
2. 打开对应 spec 文件,**整份**复制粘贴到 Claude 或 ChatGPT 的新对话
3. AI 会进入"项目协作模式",根据 spec 的阶段定义引导你执行
4. 每次新对话时把 spec 重新粘贴一次,并附上"我目前进展到第 X 阶段第 Y 步"

## 三份 spec 的设计原则

- **阶段化**:每个方向都分 3-5 个阶段,前一阶段没完成不进入下一阶段
- **可证伪**:每个阶段都有明确的"做完了"的标准,不是模糊的"差不多"
- **反自我感动**:明确列出"这个阶段不要做什么"——避免设计 logo、买域名、做品牌这类自我感动动作
- **真实预期**:每个方向都包含"如果做不起来,可能死在哪里",不画饼

## 跨方向通用原则(三份 spec 都遵循)

1. **不改 Horizon 代码**:护城河不在工程,在配置 + taste + 关系
2. **先验证再投入**:任何花钱的动作(域名、设计、付费工具)都要在有付费用户或明确数据后才做
3. **邮件名单 > 关注数**:所有方向的核心资产是**自己手里的邮件名单**,不是平台粉丝
4. **人机混合内容质量 > 纯 AI 内容数量**:AI 帮你过滤和摘要,但**人工点评**是差异化关键
5. **每周复盘**:每周固定时间(建议周日)做 30 分钟复盘,记录数据 + 反思

## 一个不会变的提醒

工具是生产力放大器,但放大的是你的判断力,不是替代你的判断力。Horizon 让你**比同行多消化 10 倍信息**,但这只有在你**有判断力去用这些信息**时才有价值。三个方向的真正护城河,都是你自己。
