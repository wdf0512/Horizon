## [IOCCC 2025 Winners: GameBoy-Shaped Emulator and 366-Byte Linux VM](https://www.ioccc.org/2025/) ⭐️ 9.0/10

The 29th International Obfuscated C Code Contest (IOCCC) announced its winners, featuring a GameBoy emulator whose code is shaped like the console, and a 366-byte one-instruction-set computer that can run Linux and Doom. These entries push obfuscation to new extremes, demonstrating how minimal C code can achieve sophisticated functionality and inspiring the programming community with their creativity and technical mastery. The GameBoy emulator was written by Nick Craig-Wood (creator of rclone); the 366-byte VM implements a one-instruction set computer (OISC) and can run Doom; the IOCCC guidelines explicitly permit LLM use; the contest website itself is obfuscated, making source code hard to find.

hackernews · matt_d · Jun 7, 05:47 · [Discussion](https://news.ycombinator.com/item?id=48432199)

**Background**: The IOCCC, first held in 1984, is an annual contest celebrating creative C obfuscation, with entries judged anonymously on cleverness and unexpected behavior. A one-instruction set computer (OISC) is an abstract machine that uses a single instruction yet can be Turing complete, showing that a minimal processor can theoretically compute anything.

## [AMD Developing New Platform with Up to 192 GB Unified Memory for Local AI](https://www.ithome.com/0/961/102.htm) ⭐️ 9.0/10

AMD announced its upcoming Ryzen AI MAX 400 Series chips will support up to 192 GB of unified memory, with 160 GB accessible by the GPU, enabling local execution of large language models with over 300 billion parameters. This dramatically expands the memory available to consumer-grade GPUs, potentially changing how AI engineers deploy large models locally by removing the need for expensive, high-end server hardware. The unified memory architecture allows CPU and GPU to share a single memory pool, reducing data transfer overhead. The Ryzen AI MAX 400 Series is likely aimed at workstations and AI development, but it is unclear if it will be used in gaming processors. AMD also noted that NVIDIA's RTX Spark's similar approach validates this direction.

telegram · zaihuapd · Jun 7, 08:32

**Background**: Unified Memory Architecture (UMA) means CPU and GPU share the same physical memory, eliminating data duplication and transfer bottlenecks. Apple's M-series chips also use UMA, achieving high bandwidth and efficiency. In traditional PC architectures, CPU and GPU have separate memory pools, requiring data copying. AMD's move brings UMA to x86 high-performance platforms, potentially enabling large local AI workloads without expensive professional GPUs like NVIDIA's A100 or H100.

The discussion reflects the growing emotional and professional toll AI inflicts on engineers, as LLMs move from simple code generation to complex domain-specific reasoning, raising urgent questions about job security and the evolving nature of technical expertise. ## [Lathe: Hands-on, Source-Backed Tutorials Generated by LLMs for Self-Learning](https://github.com/devenjarvis/lathe) ⭐️ 8.0/10

Lathe is a new Go CLI tool that uses LLM agents (Claude Code / Cursor / Codex) to generate interactive, source-backed tutorials for any technical topic. Users must read and type the code themselves in a local browser UI, with features like table of contents, side-notes, and exercises. It flips the typical LLM use-case from generating answers to fostering deep understanding, enabling self-learners to master niche technical domains where no good human-written tutorials exist. This approach combats passive learning and promotes active engagement with code. Lathe works as a CLI with agentic coding tools, can verify tutorial correctness with an LLM, and allows extensions. The creator notes output isn't perfect but errors are caught during manual typing; it has been tested primarily on macOS with Claude Code.

hackernews · devenjarvis · Jun 7, 11:16 · [Discussion](https://news.ycombinator.com/item?id=48433756)

**Background**: Large Language Model (LLM) agents are AI programs that can use tools, execute commands, and generate code. Claude Code is a popular agentic coding tool from Anthropic that integrates into terminals and IDEs. 'Vibecoding' refers to low-risk, experimental coding with AI assistance. A '3D slicer' is software that converts 3D models into printer instructions.
