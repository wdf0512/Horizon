An article catalogs recognizable stylistic patterns, or 'smells,' in LLM-generated writing, including overused phrases like 'honest caveat:' and rhetorical devices such as contrastive negation. The community discussion further surfaces additional tells and strategies to avoid LLM sameness. As AI-generated content proliferates, recognizing these tells becomes crucial for maintaining authenticity in writing, preventing plagiarism, and enhancing content quality. It also helps writers use LLMs more effectively as tools rather than crutches. Specific smells identified include lead-in phrases like 'The honest caveat:' and 'The smoking gun:', contrastive negation ('It's not X, it's Y'), and stylistic uniformity. The article also notes that LLM writing may appear superior in areas where the reader lacks expertise, making evaluation tricky.

The concept of 'smells' borrows from software engineering, where 'code smells' indicate potential deeper issues. This article extends that metaphor to writing, offering a checklist of common AI-generated traits. The phenomenon is part of a broader discourse on AI detection and preserving human voice in writing.

## [DOMD: A 20 KB Kernel Markdown WYSIWYG Editor Built from Scratch](https://github.com/do-md/domd) ⭐️ 8.0/10

DOMD is a new open-source Markdown WYSIWYG editor with a rendering engine built entirely from scratch, achieving a core size of only 20 KB (gzipped, no runtime dependencies besides React). It offers local-first, high-performance editing even for 1 MB documents, and supports AI model streaming via a CLI. By avoiding frameworks like ProseMirror, Slate, and Lexical, DOMD remains extremely lightweight while maintaining stability for large files and AI streaming, offering an alternative design pattern for frontend engineers and AI tool developers. The editor uses a single data source with immutable state, so typing, undo/redo, AI streaming injection, and chunk loading of large files all go through the same state change. The macOS version is built with Tauri and includes a Quick Look extension; the web version needs no uploads.

telegram · zaihuapd · May 28, 05:48

**Background**: Typical Markdown editors rely on rich-text frameworks like ProseMirror (a battle-tested core used by many editors) for document model and cursor management. DOMD built its own engine from scratch. Tauri is a framework for creating lightweight desktop applications using web technologies. The local-first principle keeps all data on the user’s device without accounts or cloud sync.
